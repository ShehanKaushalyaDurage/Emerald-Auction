## 📌 Overview
**Emerald Auction** is a **distributed, real-time auction platform** built using **Jakarta EE (J2EE)** technologies. It demonstrates enterprise-level backend engineering concepts such as **event-driven architecture, concurrency control, and real-time communication**.

Unlike traditional applications, this system **does not use a database**. Instead, it relies on **in-memory data structures**, allowing the focus to remain on **system design, performance, and real-time behavior**.

---

## 🎯 Why This Project Stands Out

- Designed a **real-time distributed system** without database dependency  
- Implemented **JMS-based publish/subscribe architecture**  
- Built a **concurrency-safe bidding engine**  
- Developed an **auto-bidding system with fairness guarantees**  
- Delivered **live updates using WebSockets**
